GOAL:
Write a website that converts F to C and vice versa

Pseudocode

* When the program is loaded, both F and C values are 0.
* User enters a temperature either in F or C in the form of a number.
* User presses the "submit" button.
* Program determines if it should convert F to C by C to F based on which number changed.
    * If F changed:
        * Save the F value.
        * Convert F to C.
        * Display the C value.
    * If C changed.
        * Save the C value
        * Convert C to F.
        * Display the C value.
